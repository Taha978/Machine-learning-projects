Model selection and fitting¶

Selective loop¶

We will run a loop that fits an ARIMA model for different values of p and q and selects the best model, that has the least AIC¶

We will run this loop twice: the first one will fit a model with the least squares estimation method, and the second one with the maximum likelihood estimation method.¶

Then, we will select the one with the lowest AIC¶

In [20]:
from statsmodels.tsa.arima_model import ARIMA
In [21]:
def evaluate_model_css(dataframe):
    i=0
    order_frame={'Order of AR':[],'Order of MA':[],'AIC':[]}
    order_frame=pd.DataFrame(order_frame)
    for p in range(0,4):
        
        for q in range(0,4):
            i=i+1
            model= ARIMA(dataframe,order=(p,1,q))
            results_ARIMA=model.fit(disp=1,method='css')
            order_frame.loc[i]=[p,q,results_ARIMA.aic]
    
    order_frame=order_frame.sort_values(by='AIC', ascending=True)
    print(order_frame)
